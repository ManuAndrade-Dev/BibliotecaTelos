# BibliotecaTelos

Esse projeto é um simulador de biblioteca feito em SQL Server. Ele foi criado para treinar comandos de banco de dados, como criação de tabelas, inserção de dados, consultas, funções e procedimentos.

## O que tem no projeto

- Criação do banco de dados
- Tabelas para livros, usuários e empréstimos
- Inserção de dados de exemplo
- Comandos para registrar empréstimos e devoluções
- Relatórios simples
- Funções e procedimentos armazenados
- Validações básicas

## Como usar

1. Execute os arquivos na ordem: de `01_create_db.sql` até `08_validations.sql`.
2. Use o SQL Server ou o DBeaver para rodar os scripts.
3. Os dados de exemplo já estão incluídos para facilitar os testes.
